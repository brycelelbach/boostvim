Installation instructions
=========================

Vundle
------
Install using `vundle <https://github.com/gmarik/Vundle.vim>`_ by adding

    Plugin 'hia3/boostvim'

to *.vimrc* and run *:PluginInstall*. Reload vim afterwards.
